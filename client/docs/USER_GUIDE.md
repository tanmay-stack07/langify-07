# User Guide: Multilingual Speech Web Application

## Getting Started
1. **Launch**: Open the landing page and click "Start Live Pulse".
2. **Setup**: Select your target language (default is English).
3. **Record**: Click "Ignite Session" and speak.

## The Jarvis HUD
- **Left Panel**: Real-time transcript stream. Original text and translations will appear as you speak.
- **Right Panel**: AI metrics and the Jarvis Loader. 
    - **Pulsing**: Detecting language.
    - **Spinning**: Synthesizing audio.
    - **Glowing**: Aligning translation.

## Tips for Best Results
- **Clear Audio**: Use a quiet room for high-accuracy Whisper detection.
- **Natural Speech**: You can use code-switched languages (e.g., Hinglish); the AI is tuned for nuance.
