Each phase lasts 2200ms. The cycle communicates exactly what the AI pipeline is doing: Phase Cycle — Live Translation Processing Phase 1 (0ms — 2200ms): text = "Analyzing" // Audio received, Whisper-1 intake Phase 2 (2200ms — 4400ms): text = "Processing" // Language auto-detection active Phase 3 (4400ms — 6600ms): text = "Translating" // GPT-4o translation call Phase 4 (6600ms — done): text = "Finalizing" // Result preparation + display // If API returns before cycle completes: stop at current phase // If API takes longer: loop back to Phase 1 7.4 JarvisLoaderOverlay — Wrapper Structure The raw AiLoader component is always wrapped inside JarvisLoaderOverlay, which adds the HUD, scan line, corner brackets, data bars, and grid lines. Never use AiLoader directly — always use JarvisLoaderOverlay. Design Decision Summary Decision Rationale ai-loader mandatory on Live Translate It is the visual identity of the Universal Agent. Its absence makes the feature feel ordinary. JarvisOverlay wraps AiLoader always Never use AiLoader directly — the wrapper adds the HUD, corners, scan, and phase cycling that make it feel intelligent. ProcessingCard inline, not fullscreen overlay Fullscreen overlay for each utterance would be too disruptive in a continuous session. Inline card is the right balance. Phase labels: Analyzing → Processing → Translating → Finalizing Communicates the exact AI pipeline stages. More meaningful than a spinning wheel. text prop cycles every 2200ms Fast enough to feel dynamic, slow enough to be readable. Letter animation at 3s cycle is comfortable. One target language setting only The Universal Agent concept. Source language is auto-detected — adding a source selector destroys the magic. Supabase save at session end only Single batch insert avoids 50+ API calls during active session. Clean, efficient, professional. Vue.js frontend Reactive state handles complex UI transitions of the Live Translate session cleanly without framework friction. Dark theme on Live Translate only Dark signals intelligence. The Jarvis palette is reserved for the AI-heaviest feature to maintain its visual impact. DownloadPanel with SRT + VTT Subtitle export is what makes this more than a demo. It is the single most useful concrete output for professionals.
